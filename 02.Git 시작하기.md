## 5. add

* `git`에서 커밋할 대상 파일을 `staging area` 로 이동시키는 명령어이다.

  ```
  $ git add test.txt --- 특정 파일을 stage
  $ git add image/   --- 특정 폴더를 stage
  $ git add .        --- 모든 파일 및 폴더 stage
  ```
