# Git 설치하기

> Git은 분산형버전관리시스템(DVCS)이다.
>
> 깃의 핵심기능은 버전 관리(Version Control), 백업(Backup), 협업(Collaboration)이다.

## 1. [Git 사이트](https://git-scm.com/) 접속

* 오른쪽 하단의 Download ~를 클릭

* 다음 화면으로 이동하면서 자동으로 파일을 내려받기 시작한다.
